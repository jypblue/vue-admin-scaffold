# vue-admin-scaffold

> this is a vue admin scaffold for [vue-cli]

### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

### Browser Support

Modern browsers and Internet Explorer 11+.
